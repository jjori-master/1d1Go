#### UNIT 34 채널 사용하기

> 채널은 고루틴낄 데이터를 주고 받고,  실행 흐름을 제어하는 기능
>
> 모든 타입을 채널로 사용 가능
> 채널 자체는 값이 아닌 레퍼런스 타입



- 모든 타입을 채널로 사용가능

  ```go
  c := make(chan int) // int형 채널 생성
  
  var c2 chan int
  c2 = make(chan int) // 채널은 사용하기전 반드시 make 함수로 공간 할당
  ```
